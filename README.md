# music-numeral-system

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A counting song for the Japanese system of large numbers (命数法, *meisūhō*), generated by AI. This project provides a web-based music player to listen to the song and data files detailing the numeral systems.

## Demo

**[日本の数え歌 - 命数法 (Japanese Counting Song - Numeral System)](https://code4fukui.github.io/music-numeral-system/)**

The web application features a music player with a playlist of different "minimal techno" style tracks, album art, and synchronized lyrics.

## About the Song

The song's lyrics consist of the Japanese names for large number units, from one (一) to an immeasurable number (無量大数). It serves as a mnemonic device to help memorize this sequence. The music was generated using [Suno AI](https://suno.com/).
