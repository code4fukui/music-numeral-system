# music-numeral-system

日本の命数法（大数の体系）をテーマにした、AI生成による数え歌です。本プロジェクトでは、楽曲を視聴できるウェブベースの音楽プレイヤーと、命数法の詳細をまとめたデータファイルを提供しています。

## デモ

**[日本の数え歌 - 命数法 (Japanese Counting Song - Numeral System)](https://code4fukui.github.io/music-numeral-system/)**

このウェブアプリケーションは、様々な「ミニマル・テクノ」スタイルの楽曲のプレイリスト、アルバムアート、および同期された歌詞を備えた音楽プレイヤーです。

## 曲について

歌詞は、一から無量大数までの日本の大数の単位名で構成されており、この順序を覚えるための暗記歌となっています。楽曲は[Suno AI](https://suno.com/)を使用して生成されました。
